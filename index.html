This tool stores camper data in Google Firebase. Before campers can use it, the camp director needs to configure the connection.

For camp directors: open this HTML file in a text editor and find the section near the top of the <script> block labeled FIREBASE CONFIGURATION. Replace the placeholder values with your Firebase project's web config.

Quick Setup Outline

  1. Create a free Firebase project at console.firebase.google.com
  2. Enable Firestore Database (start in test mode)
  3. Enable Anonymous Authentication under Authentication → Sign-in method
  4. Add a Web App in Project Settings and copy the firebaseConfig values
  5. Paste those values into this HTML file, then host the file (e.g., GitHub Pages)
  6. Apply the recommended Firestore security rules (see setup guide)

Once configured, this screen will be replaced with the home screen automatically.
